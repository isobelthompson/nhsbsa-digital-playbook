## Power BI Data Analyst Associate Exam PL-300  
  
### Content  
  
- Prepare the data
- Model the data
- Visualize and analyze the data
- Deploy and maintain assets  
  
### Roles/Teams: Mandatory  
  
- Data Modellers
- Data Visualisers: BI/MI teams
- Data Scientists  
  
### Roles/Teams: Optional  
  
- Data Analysts
- ETL Developers
